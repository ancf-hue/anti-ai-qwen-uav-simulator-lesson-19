## 📖 О проекте

**«Кинооператор — Аэросъёмка»** — это интерактивный 3D-симулятор, созданный в рамках бесплатного социального офлайн-проекта **Anti-AI Qwen Academy**. Тренажёр позволяет освоить основы управления беспилотным летательным аппаратом (БПЛА) и навыки профессиональной аэросъёмки в безопасной виртуальной среде.

### 🎯 Цель тренажёра

Научить будущих кинооператоров:
- Управлять дроном в трёхмерном пространстве
- Выполнять плавные манёвры и удерживать стабильный кадр
- Работать с камерой на 360°
- Планировать траекторию полёта с учётом препятствий
- Соблюдать правила безопасности при съёмке

## 🕹️ Управление

### Клавиатура
| Клавиша | Действие |
|---------|----------|
| `W` / `↑` | Движение вперёд |
| `S` / `↓` | Движение назад |
| `A` / `←` | Движение влево |
| `D` / `→` | Движение вправо |
| `Q` / `Shift` | Набор высоты |
| `E` / `Ctrl` | Снижение |

### Мышь
- **Зажать ЛКМ + двигать** — обзор камеры на 360°

### 🎯 Правила съёмки
1. Наведите камеру на целевой объект (розовая сфера)
2. Удерживайте объект в кадре **3 секунды**
3. Избегайте столкновений с препятствиями
4. Не допускайте падения (высота < 2.5 м)
